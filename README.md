# JsBlog
- 自己纯 Js 写的 前后端博客
## 博客功能
- 博客文章: 发布, 删除, 修改
- 评论: 发布, 删除, 修改
## 说明
- 自己实现了 JQuery 中的部分功能的库
- 自己实现的 ajax 与后端的数据交互
- 利用 Json 作为数据库
- 自己写的 module 类进行数据交换
